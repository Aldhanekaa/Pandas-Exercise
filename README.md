# Pandas-Exercise
pandas exercise

# How to use it

  fork this repo

  clone the repo that you fork

# Open Jupyter notebook
  first of all you must install jupyter notebook
  in windows:
  
  ```pip install jupyterlab```

  in Linux:
  
  ```pip3 install jupyterlab```

  and then open jupyter notebook with
  
  ```jupyter-notebook```
